# 🏛️ Vaishali Heritage – Smart Tourism Platform

## 📖 Overview

**Vaishali Heritage** is a full-stack tourism platform designed to help visitors explore the rich cultural and historical heritage of **Vaishali, Bihar**, one of the world's oldest republics and an important site in both Buddhism and Jainism.

The platform provides detailed information about historical places, intelligent trip planning, AI-powered travel assistance, reviews, and route planning through a modern web application.

---

# ✨ Features

### 🏛️ Explore Heritage Places

* Browse famous tourist attractions in Vaishali
* View images and detailed descriptions
* Learn historical significance
* Ratings and visitor information

### 📍 Place Details

* Complete information about every monument
* History
* Best time to visit
* Distance
* Visitor ratings
* Related places suggestions

### 🤖 AI Tour Guide

* Interactive AI chatbot
* Answers questions related to Vaishali's history, culture, monuments, and travel
* Designed to behave like a local tour guide
* Backend integrated with external AI APIs (currently under development)

### 🗺️ Smart Trip Planner

* Generate personalized travel itineraries
* Suggest places based on available time

### 🛣️ Route Planner

* Plan efficient travel routes between attractions

### ⭐ Reviews & Ratings

* Users can share travel experiences
* Community-based ratings

### 👨‍💼 Admin Dashboard

* Manage heritage places
* Manage reviews
* Manage users
* Dashboard statistics
